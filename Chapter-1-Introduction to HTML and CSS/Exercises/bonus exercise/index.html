HTML & CSS QUICK REFERENCE GUIDE

HTML Reference Guide

HTML stands for HyperText Markup Language.It is the standard language used to create the structure and content of web pages.Think of HTML as the skeleton of a webpage. It tells the browser:

  • What to display (text, images, links, etc.).
  • How content is organized (headings, paragraphs, lists, etc.).
  • Where elements go on the page.

10 HTML Tags

  1. html:Root element of an HTML document.
  2. head:Contains meta info like title, styles, links.
  3. title:Sets the title shown in the browser tab.
  4. body:Contains everything that's visible on the page.
  5. h1 to h6:Headings (h1 is largest, h6 is smallest).
  6. p:Paragraph.
  7. a: Anchor tag (used for links).
  8. img:Image element.
  9. div:Generic container for layout and styling.
  10. span:Inline container (often used for styling text parts).

CSS Reference GUIDE

CSS stands for Cascading Style Sheets.It is used to style and design the HTML content.Think of CSS as the clothes and colors that make the skeleton (HTML) look nice. It controls:

  • Layout
  • Colors
  • Fonts
  • Spacing
  • Animations
  • Responsiveness (how it looks on mobile vs. desktop)

8 CSS Tags

  1. color:Sets the text color.
  2. background-color:Sets the background color of an element.
  3. font-size:Controls the size of text.
  4. margin:Space outside the element (like padding around it).
  5. padding:Space inside the element (like cushion for content).
  6. border:Adds a border around elements.
  7. display:Controls how elements are displayed.
